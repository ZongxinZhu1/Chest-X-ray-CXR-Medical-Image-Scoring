Scoring Guideline Summary

  • 2 Points: Perfect Match The synthetic image is a perfect match for the text prompt, with all pathologies and details accurately represented.
  • 1 Point: Good Match The synthetic image displays most of the pathologies/details mentioned in the text prompt, with no core errors.
  • 0 Point: Core Only The synthetic image only displays the core pathology of the text prompt, without all details reflected.
  • -1 Point: Omissions The synthetic image fails to display all pathologies mentioned in the text prompt, with obvious omissions.
  • -2 Points: Mismatch/Error The synthetic image is completely mismatched with the text prompt, with no corresponding pathologies or serious errors.
