Scoring Guide

Use the following rubric when assigning scores to each image against the report text.
2
The synthetic image is a perfect match for the text prompt, with all pathologies and details accurately represented.
1
The synthetic image displays most of the pathologies/details mentioned in the text prompt, with no core errors.
0
The synthetic image only displays the core pathology of the text prompt, without all details reflected.
-1
The synthetic image fails to display all pathologies mentioned in the text prompt, with obvious omissions.
-2
The synthetic image is completely mismatched with the text prompt, with no corresponding pathologies or serious errors.
Please directly record the score for each sample set using the panel on the right.
